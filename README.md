## 🎮 Story Preview

**The Setup**: Strange Dark Fountains have begun appearing throughout the Underground, transforming familiar locations into chaotic Dark Worlds. The balance between Light and Dark is being destroyed. Frisk has not fallen yet... but destiny calls.

### Chapter 1: The Dark Awakening
- 🔥 Toriel senses the disturbance
- 💀 Sans knows something is wrong
- 🌑 A Dark Fountain erupts in the Ruins
- ⚔️ The prophecy awakens

## 🚀 Getting Started

### Prerequisites
- **Godot 4.2+** ([Download](https://godotengine.org/download))

### Setup
```bash
git clone https://github.com/mkldev011/undertale-dark-underground.git
cd undertale-dark-underground
# Open project.godot in Godot 4.2+
# Press F5 to run
```

### Controls
| Action | Keyboard |
|--------|----------|
| Move | WASD / Arrows |
| Interact | Enter / Space / E |
| Confirm | Enter / Space |
| Cancel | Escape |
| Pause | Tab |
